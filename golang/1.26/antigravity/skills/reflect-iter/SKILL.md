# Go 1.26 Reflection Iterators

Expert at using the new iterator-based methods in the `reflect` package for efficient inspection of types and values.

## Purpose
- Iterate over fields of a struct.
- Iterate over methods of a type or value.
- Iterate over input and output parameters of a function.

## Key APIs
- `Type.Fields()`, `Type.Methods()`, `Type.Ins()`, `Type.Outs()`
- `Value.Fields()`, `Value.Methods()`
