# Docsy General Skill

Expert at managing Docsy theme components and documentation content.

## Capabilities
- **Environment Validation**: Verify that the environment has Hugo "extended", Node.js, and PostCSS installed.
- **Content Management**: Create and review documentation pages following Docsy style and front matter rules.
- **Shortcode Usage**: Correctly apply Docsy-specific shortcodes like `card`, `tabpane`, and `mermaid`.

## Usage
Use this skill when:
- Setting up or verifying a local Docsy development environment.
- Adding or modifying documentation content.
- Troubleshooting build issues related to SCSS or PostCSS.

## Scripts
- `check-env.sh`: Checks for Hugo extended, Node.js, and PostCSS.
